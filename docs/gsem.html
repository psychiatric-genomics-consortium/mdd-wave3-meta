Structure Meta-Analysis with GenomicSEM

Summary statistics were grouped based on phenotype, then meta-analysed:

  • clin: Clinical assessment
  • ehr: Electronic health records
  • quest: Questionnaire
  • self: Single-item self report

The LDSC covariance structure of the four MDD phenotype was calcuted using GenomicSEM.

GWAS-by-subtraction

Fit a GWAS-by-subtraction to decompose variance into that of minimally-phenotyped depression shared with maximal phenotypes versus specific to maximal phenotypes.

md.model <- "MD =~ NA*self + clin + ehr + quest
MDD =~ NA*clin + ehr + quest

MD ~~ 1*MD
MDD ~~ 1*MDD
MD ~~ 0*MDD

self ~~ 0*clin + 0*ehr + 0*quest
self ~~ 0*self"
md.fit <- usermodel(covstruc=mdd_covstruct, estimation='DWLS', model=md.model)
